# Bitasmbl-AI-Powered-Study-Notes-Generator-5a04bc

## Description
Build a web application that allows students to input lectures, textbooks, or articles and automatically generate concise, structured study notes. The system uses AI summarization and keyword extraction to create easy-to-review content while maintaining readability and context.

## Tech Stack
- Go
- Objective-C
- Bootstrap

## Requirements
- Go
- Objective-C
- Bootstrap

## Installation
bash
git clone https://github.com/he1snber8/AI-Powered-Study-Notes-Generator.git
cd AI-Powered-Study-Notes-Generator


## Usage
- Build and run the Go backend.
- Build and run the Objective-C client using Bootstrap UI.

## Implementation Steps
1. Define Go services for summarization and keyword extraction.
2. Implement HTTP handlers in Go for text input and notes generation.
3. Build Objective-C UI integrating Bootstrap for layout.
4. Connect Objective-C client to Go endpoints.
5. Render generated notes in a structured format.
